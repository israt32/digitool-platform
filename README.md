⚛️ Digitool Platform

📌 Project Overview
DigiTool-Platform is a modern web application that provides users with a suite of digital tools to boost their careers, build resumes, track job applications, optimize LinkedIn profiles, and create stunning portfolios. Whether you're a fresh graduate, a freelancer, or a professional, DigiTool-Platform has the right tool to help you succeed.

🛠️ Technologies Used
⚛️ React.js – For building a responsive and interactive UI
🎨 Tailwind CSS – For modern and mobile-friendly styling
🖌️ React Icons – For beautiful and scalable icons
💻 JavaScript (ES6+) – For functionality and interactivity
🔀 React Router – For navigation across pages

✨ Key Features
🧰 Career Tools Suite – Includes Resume Builders, Portfolio Creators, Job Trackers, and Interview Prep Kits.
🚀 Smart Optimization – LinkedIn optimizer, ATS-friendly resumes, and SEO-friendly portfolios to enhance your professional visibility.
💰 Subscription & Pricing Options – Flexible pricing: one-time, monthly, and yearly plans to match every user’s needs.
