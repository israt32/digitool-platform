📌 Project Overview
Digi-Tools is a sleek web app where users can discover and subscribe to a variety of AI tools. It offers a smooth, interactive experience for browsing tools, managing the cart, and viewing real-time subscription totals. The design emphasizes a clean, intuitive interface and effortless usability.

🚀 Technologies Used
⚛️ React – For building dynamic UI components
🎨 Tailwind CSS – For responsive and modern styling
🌼 DaisyUI – For pre-built, beautiful UI elements

✨ Key Features
🛒 Add subscriptions – Users can easily add AI tools to their cart (dummy purchase system)
❌ Remove items – Quickly delete selected subscriptions from the cart
💰 Real-time pricing – Instantly see the total cost as items are added or removed
